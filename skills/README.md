# skills/

Higher-level **playbooks / workflows** (markdown skills) that orchestrate one or
more MCP servers to accomplish a SOC task end-to-end — for example, triaging an
alert, enriching an IOC across MISP and TheHive, or building a posture summary
for a CISO.

Skills compose the read tools exposed by `servers/` and present results through
the persona renderers in `core/renderers/`.
